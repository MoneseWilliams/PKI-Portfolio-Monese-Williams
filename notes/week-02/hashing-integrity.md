Document the following in your Week 2 lab notes:
- The hash changed after a small modification because hashing is designed so that any minor change in the data will create a different output. This helps allow systems to know wether or not data that has been transmitted has been altred while in transmission
- Hashing does not provide confidentiality because it doesnt encrypt the data and protect it, its used for checking wether or not data has been modified
- The security property hashing provides is integrity
- Hashing is used in PKI systems to make sure data has not been changed. For example, the digital certificates that are issued by the CA signs the certificate and the secure systems verifies using hashing and the digital signatures to make sure no signatures are altered
